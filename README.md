# 🏗️ ARCHITECT v9.05 | Terminal Management System

**ARCHITECT** je výkonný, minimalistický nástroj pro správu projektů a sledování deadlinů běžící přímo v terminálu. Je navržen pro uživatele, kteří vyžadují **čistý kód**, **industriální estetiku** a **maximální přehlednost** bez zbytečného klikání v GUI.

## 💎 Klíčové vlastnosti systému

* **Deep Amber Visuals:** Sjednocené jantarové barevné schéma (ANSI 33) inspirované monochromatickými monitory 80. let.
* **Stationary UI:** Optimalizované vykreslování. Logo zůstává fixní na vrcholu relace, data plynule proudí pod ním.
* **Persistent Security:** Každý operátor má vlastní profil chráněný reverzním Access Key mechanismem.
* **Inteligentní Time-Analysis:** Automatická kontrola termínů (🔴 EXPIROVÁNO, 🟡 VAROVÁNÍ).
* **Dynamic Database Management:** Interaktivní přidávání i selektivní mazání záznamů přímo z konzole.
